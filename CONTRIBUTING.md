## Contributing

Thanks for your interest in improving **tinyFolder**.

### Quick workflow

1. Fork the repo
2. Create a branch:

```bash
git checkout -b feature/my-change
```

3. Install deps and run checks:

```bash
cd gzmo-daemon
bun install
bun run typecheck
```

4. Commit and open a pull request

### What’s most helpful

- Fix docs, broken paths, and onboarding friction
- Improve reliability of the folder watcher + task routing
- Add tests around regressions and vault edge-cases
