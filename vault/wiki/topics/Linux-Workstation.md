# Linux Workstation

The 2026 sovereign Linux desktop — a minimalist, telemetry-free development environment designed for maximum productivity with AI-native tooling.

## Architecture

- **OS:** Ubuntu Linux
- **IDE:** VSCodium (telemetry-free VS Code)
- **AI Tooling:** Local LLMs via LM Studio + llama.cpp
- **Agent:** [[GZMO]] via [[OpenClaw]]
- **Knowledge:** Obsidian vault with [[LLM-Wiki]] pattern
- **Containerization:** Docker for dev environments

## Design Principles

- Minimalist desktop — function over aesthetics
- No proprietary dependencies
- All AI inference local-first (see [[Sovereign-AI]])
- Overclocking for maximum hardware utilization
