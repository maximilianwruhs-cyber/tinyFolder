# Dreams — Identity Evolution Proposals

This is GZMO's reflection space. Each dream is a proposal to evolve `SOUL.md`, `AGENTS.md`, or any core identity file.

**GZMO writes proposals here. The User decides.**

## Workflow

1. GZMO observes patterns during wiki maintenance or conversations
2. GZMO creates a dream page: `dreams/YYYY-MM-DD-<topic>.md`
3. User reviews in Obsidian (or GZMO highlights during conversation)
4. User sets status:
   - `proposed` → Under review
   - `approved` → Ready to merge into target file
   - `merged` → Done, change is live
   - `rejected` → Not adopted (with reason)
